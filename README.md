# Wealth Dashboard

부동산을 제외한 순자산 기준으로 올해 자산 목표 달성률을 관리하는 개인용 웹앱입니다.

## 기준

- 올해 목표 금액: 180,000,000원
- 목표용 순자산: 현금 + 주식 + 연금 + 금 - 대출
- 부동산: 목표 달성률에서 제외하고 참고 자산으로만 표시
- 업데이트 주기: 주 1회

## 개인정보

입력한 자산 데이터는 서버나 GitHub에 저장되지 않고, 접속한 브라우저의 로컬 저장공간에만 저장됩니다.

PC와 휴대폰은 서로 다른 브라우저 저장공간을 사용하므로 데이터가 자동 동기화되지 않습니다.
